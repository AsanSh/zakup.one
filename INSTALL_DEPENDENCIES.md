# Установка зависимостей на Spaceship

## ✅ Что уже сделано:

1. ✅ `.env` файл создан и загружен на сервер с безопасным SECRET_KEY
2. ✅ Скрипт `setup.sh` загружен на сервер
3. ✅ Все файлы проекта загружены

## 📋 Установка зависимостей

### Вариант 1: Через SSH терминал (если доступен)

Подключитесь к серверу через SSH и выполните:

```bash
cd zakup.one
chmod +x setup.sh
./setup.sh
```

Или вручную:
```bash
cd zakup.one
pip install -r requirements.txt
```

### Вариант 2: Через панель управления Spaceship

1. Зайдите в панель управления Spaceship
2. Найдите раздел "Terminal" или "SSH Access"
3. Выполните команды:
   ```bash
   cd zakup.one
   pip install -r requirements.txt
   ```

### Вариант 3: Через встроенный терминал в панели

Многие хостинги предоставляют встроенный терминал в веб-интерфейсе:
1. Зайдите в панель управления
2. Найдите "Terminal" или "Command Line"
3. Выполните установку зависимостей

## 🔧 Настройка Python приложения

В панели управления Spaceship найдите настройки Python приложения:

1. **Entry Point (точка входа):** `wsgi:application`
2. **Рабочая директория:** `zakup.one` (или полный путь)
3. **Версия Python:** 3.11 или доступная (обычно 3.9+)
4. **Переменные окружения:** Убедитесь что `.env` файл читается

## ⚠️ Если нет доступа к терминалу

Если у вас нет доступа к SSH или терминалу:
1. Обратитесь в поддержку Spaceship
2. Попросите их установить зависимости из `requirements.txt`
3. Или используйте встроенные инструменты панели управления для установки пакетов Python
