## Технические детали

### Backend API endpoints (требуется добавить):
- `GET /api/v1/admin/stats` - статистика для Dashboard
- `GET /api/v1/admin/users/pending` - заявки на регистрацию
- `POST /api/v1/admin/users/{id}/approve` - одобрить заявку
- `POST /api/v1/admin/users/{id}/reject` - отклонить заявку
- `GET /api/v1/admin/orders/stats` - статистика по заявкам
- `GET /api/v1/admin/settings` - системные настройки
- `PUT /api/v1/admin/settings` - обновить настройки

### Frontend компоненты (требуется создать):
- `DataTable.tsx` - универсальная таблица с пагинацией
- `PriceImportModal.tsx` - модальное окно импорта цен
- `OrderDetailModal.tsx` - детальный просмотр заявки
- `StatsCard.tsx` - карточка статистики
- `Chart.tsx` - компонент для графиков
