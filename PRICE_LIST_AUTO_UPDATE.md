# Автоматическое обновление прайс-листов

## Возможности

1. **Скачивание прайс-листа по URL** - администратор может указать URL для автоматического скачивания
2. **Автоматическое обновление** - система может автоматически скачивать и обновлять прайс-листы по расписанию:
   - Ежедневно
   - Еженедельно
   - Ежемесячно
   - Вручную

## Использование

### 1. Добавление прайс-листа по URL

1. Перейдите в админ-панель → **Прайс-листы**
2. Нажмите кнопку **"Скачать по URL"**
3. Заполните форму:
   - Выберите поставщика
   - Укажите URL для скачивания (например: `https://stroydvor.kg/wp-content/uploads/прайс-лист-10.11.25-.xlsx`)
   - Выберите частоту обновления
   - При необходимости укажите строку заголовка и строку начала данных
4. Нажмите **"Скачать и импортировать"**

### 2. Управление автоматическими обновлениями

1. Перейдите в админ-панель → **Прайс-листы** → **Обновление прайс-листов**
2. В таблице отображаются все настроенные автоматические обновления
3. Для каждого обновления доступны действия:
   - **Изменить** - изменить частоту обновления и статус
   - **Запустить сейчас** - запустить обновление вручную
   - **Активировать/Деактивировать** - включить или выключить автоматическое обновление

## Парсер для Стройдвор

Система автоматически определяет, что URL относится к Стройдвор (`stroydvor` в URL), и использует специальный парсер `StroydvorParser`, который:
- Определяет категории товаров
- Парсит название, цену и единицу измерения
- Импортирует товары в базу данных

## Настройка базы данных

Перед использованием необходимо создать таблицу для автоматических обновлений:

```sql
-- Выполните SQL скрипт
\i create_price_list_updates_table.sql
```

Или используйте Alembic для создания миграции.

## API Endpoints

- `POST /api/v1/admin/price-lists/download-and-import` - скачать и импортировать прайс-лист
- `GET /api/v1/admin/price-lists/updates` - получить список обновлений
- `PUT /api/v1/admin/price-lists/updates/{update_id}` - обновить настройки
- `POST /api/v1/admin/price-lists/updates/{update_id}/run` - запустить обновление вручную

## Автоматическое выполнение обновлений

Для автоматического выполнения запланированных обновлений необходимо настроить cron job или scheduled task, который будет вызывать метод `process_scheduled_updates()` из `PriceListDownloader`.

Пример (Python):
```python
from app.core.database import SessionLocal
from app.services.price_list_downloader import PriceListDownloader

db = SessionLocal()
downloader = PriceListDownloader(db)
result = downloader.process_scheduled_updates()
db.close()
```
