# Запуск фронтенда

## Проблема: "не работает"

Если фронтенд не запускается, выполните следующие шаги:

### 1. Установите зависимости

```bash
cd frontend
npm install
```

Это установит все необходимые пакеты (React, Vite, Tailwind и т.д.)

### 2. Запустите dev сервер

```bash
npm run dev
```

Фронтенд будет доступен на: **http://localhost:5467**

### 3. Убедитесь, что backend запущен

Backend должен быть запущен на порту 8000:

```bash
# В другом терминале
python run.py
```

### 4. Проверьте консоль браузера

Откройте DevTools (F12) и проверьте консоль на наличие ошибок.

### 5. Проверьте консоль терминала

В терминале, где запущен `npm run dev`, должны быть сообщения о запуске сервера.

## Возможные проблемы

### Ошибка: "Cannot find module"
- Решение: Выполните `npm install` в директории `frontend`

### Ошибка: "Port 5467 is already in use"
- Решение: Закройте другой процесс на этом порту или измените порт в `vite.config.ts`

### Ошибка: "CORS error"
- Решение: Убедитесь, что backend запущен и CORS настроен правильно в `app/core/config.py`

### Страница белая/пустая
- Решение: Проверьте консоль браузера на ошибки JavaScript
