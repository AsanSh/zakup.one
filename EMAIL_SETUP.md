# Настройка Email-уведомлений

## Описание

Система отправляет автоматические email-уведомления в следующих случаях:

1. **При регистрации** - клиент получает письмо о том, что заявка отправлена на рассмотрение
2. **При одобрении заявки** - клиент получает письмо об одобрении заявки
3. **При отклонении заявки** - клиент получает письмо об отклонении заявки

## Настройка SMTP

### 1. Создайте файл `.env` в корне проекта (если его еще нет)

### 2. Добавьте настройки SMTP в `.env`:

```env
# Email настройки
MAIL_USERNAME=your-email@gmail.com
MAIL_PASSWORD=your-app-password
MAIL_FROM=noreply@zakup.one
MAIL_FROM_NAME=ZAKUP.ONE
MAIL_PORT=587
MAIL_SERVER=smtp.gmail.com
MAIL_STARTTLS=True
MAIL_SSL_TLS=False
USE_CREDENTIALS=True
VALIDATE_CERTS=True
```

### 3. Для Gmail:

1. Включите двухфакторную аутентификацию в вашем Google аккаунте
2. Создайте "Пароль приложения" (App Password):
   - Перейдите в [Настройки аккаунта Google](https://myaccount.google.com/)
   - Безопасность → Двухэтапная аутентификация → Пароли приложений
   - Создайте новый пароль приложения для "Почта"
   - Используйте этот пароль в `MAIL_PASSWORD`

### 4. Для других почтовых сервисов:

**Yandex:**
```env
MAIL_SERVER=smtp.yandex.ru
MAIL_PORT=465
MAIL_SSL_TLS=True
MAIL_STARTTLS=False
```

**Mail.ru:**
```env
MAIL_SERVER=smtp.mail.ru
MAIL_PORT=465
MAIL_SSL_TLS=True
MAIL_STARTTLS=False
```

**Outlook/Office365:**
```env
MAIL_SERVER=smtp.office365.com
MAIL_PORT=587
MAIL_SSL_TLS=False
MAIL_STARTTLS=True
```

## Тестирование

После настройки SMTP, система автоматически будет отправлять письма:

1. При регистрации нового пользователя
2. При одобрении заявки администратором
3. При отклонении заявки администратором

## Важно

- Если SMTP не настроен, система продолжит работать, но письма отправляться не будут
- Ошибки отправки email логируются, но не прерывают выполнение операций
- Убедитесь, что `MAIL_USERNAME` и `MAIL_PASSWORD` правильно настроены

## Проверка работы

1. Зарегистрируйте нового пользователя
2. Проверьте почту - должно прийти письмо о регистрации
3. Одобрите заявку в админ-панели
4. Проверьте почту - должно прийти письмо об одобрении
