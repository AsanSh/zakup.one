# Конфигурация для Spaceship

## ✅ Выполнено автоматически:

1. **Создан .env файл** на сервере с настройками:
   - Сгенерирован безопасный SECRET_KEY
   - Настроен DATABASE_URL для SQLite
   - Настроены CORS_ORIGINS для домена zakup.one
   - Настроены все необходимые параметры

2. **Загружен скрипт установки** `setup.sh` на сервер

## 📋 Что нужно сделать в панели Spaceship:

### 1. Настройка Python приложения

В панели управления Spaceship найдите настройки Python приложения и укажите:

- **Entry Point (точка входа):** `wsgi:application`
- **Рабочая директория:** `zakup.one` (или полный путь к папке)
- **Версия Python:** 3.11 или доступная на сервере (обычно 3.9+)

### 2. Установка зависимостей

Выполните через SSH терминал в панели Spaceship или через встроенный терминал:

```bash
cd zakup.one
chmod +x setup.sh
./setup.sh
```

Или вручную:
```bash
cd zakup.one
pip install -r requirements.txt
```

### 3. Настройка прав доступа

Установите права на папки (через SSH или панель управления):

```bash
cd zakup.one
chmod 755 .
chmod 755 app
chmod 755 frontend
chmod 777 uploads  # Для загрузки файлов
chmod 644 *.py
chmod 644 .htaccess
chmod 644 .env
```

### 4. Перезапуск приложения

После настройки перезапустите Python приложение в панели Spaceship.

## 🔍 Проверка

После выполнения всех шагов проверьте:

1. **API Health:**
   ```
   https://www.zakup.one/api/v1/health
   ```
   Должен вернуть: `{"status": "ok"}`

2. **Главная страница:**
   ```
   https://www.zakup.one
   ```

3. **API Документация:**
   ```
   https://www.zakup.one/api/docs
   ```

## 📝 Примечания

- `.env` файл уже создан на сервере с безопасным SECRET_KEY
- Все файлы загружены в папку `zakup.one/`
- Скрипт `setup.sh` готов к выполнению для установки зависимостей

## ⚠️ Если нет доступа к SSH

Если у вас нет доступа к SSH терминалу, установку зависимостей можно выполнить через:
- Встроенный терминал в панели Spaceship
- Или обратитесь в поддержку Spaceship для установки зависимостей
