# Структура Frontend приложения ZAKUP.ONE

## Маршруты (Routes)

### 1. `/login` - Страница входа
- **Файл:** `frontend/src/pages/LoginPage.tsx`
- **Доступ:** Все пользователи
- **Описание:** Форма входа с email и паролем

### 2. `/admin` - Админ-панель
- **Файл:** `frontend/src/pages/AdminDashboard.tsx`
- **Доступ:** Только ADMIN
- **Описание:** Карточки для управления:
  - Поставщики
  - Товары
  - Заявки
  - Клиенты
- **Ссылки ведут на:** Django Admin (http://localhost:8000/admin/...)

### 3. `/customer` - Клиентский интерфейс (Каталог товаров)
- **Файл:** `frontend/src/pages/ProductsPage.tsx`
- **Доступ:** Только CLIENT
- **Описание:** 
  - Хедер с навигацией (Товары, Сборка заявки, Мои заявки)
  - Поисковая строка
  - Список товаров по категориям
  - Кнопки добавления в корзину

### 4. `/cart` - Корзина
- **Файл:** `frontend/src/pages/CartPage.tsx`
- **Доступ:** Только CLIENT
- **Описание:** Страница сборки заявки (корзина)

### 5. `/orders` - Мои заявки
- **Файл:** `frontend/src/pages/OrdersPage.tsx`
- **Доступ:** Только CLIENT
- **Описание:** История заказов клиента

## Как открыть:

1. **Админ-панель:** http://localhost:5173/admin
2. **Клиентский интерфейс:** http://localhost:5173/customer

## Примечание:

`ClientDashboard.tsx` просто редиректит на `/customer` (ProductsPage), поэтому клиентский интерфейс находится в `ProductsPage.tsx`.

## Разделение интерфейсов:

- **Клиентская часть:** `/customer` - для пользователей с ролью CLIENT
- **Админская часть:** `/admin` - для пользователей с ролью ADMIN
