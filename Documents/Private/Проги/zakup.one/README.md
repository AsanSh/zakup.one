# zakup.one - Telegram WebApp

Telegram веб-приложение для заказа строительных материалов.

## Описание

zakup.one - это мобильное веб-приложение, интегрированное с Telegram, предназначенное для заказа строительных материалов. Приложение предоставляет удобный интерфейс для просмотра каталога товаров, управления корзиной и оформления заказов.

## Функциональность

### Основные страницы:
- **Главная** - приветственная страница
- **Каталог** - просмотр товаров с возможностью добавления в корзину
- **Корзина** - управление заказами и оформление покупок
- **Заявка** - создание заявок различными способами
- **Профиль** - управление данными пользователя

### Ключевые возможности:
- Интеграция с Telegram WebApp API
- Адаптивный дизайн для мобильных устройств
- Управление корзиной покупок
- Поиск товаров
- Оформление заказов
- Уведомления через Telegram

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для тестирования в Telegram используйте BotFather для создания бота

## Telegram Bot настройка

1. Создайте бота через @BotFather
2. Установите команды:
   ```
   start - Запустить приложение
   help - Помощь
   ```
3. Настройте Web App URL: `https://zakup.one`
