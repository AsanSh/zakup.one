## 📋 Что нужно сделать в панели Spaceship:

### ШАГ 1: Установка зависимостей Python

**Вариант A - Через SSH/Терминал:**
```bash
cd zakup.one
pip install -r requirements.txt
```

**Вариант B - Через панель управления:**
1. Зайдите в панель Spaceship
2. Найдите раздел "Python" или "Applications"
3. Выберите "Install Dependencies" или используйте встроенный терминал
4. Выполните: `pip install -r requirements.txt`

### ШАГ 2: Настройка Python приложения

В панели управления Spaceship найдите настройки Python приложения:

- **Entry Point:** `wsgi:application`
- **Working Directory:** `zakup.one` (или полный путь)
- **Python Version:** 3.11 или доступная версия
- **Environment Variables:** Убедитесь что `.env` файл читается

### ШАГ 3: Настройка прав доступа

Через SSH или панель управления:
```bash
cd zakup.one
chmod 755 .
chmod 755 app
chmod 755 frontend
chmod 777 uploads
chmod 644 *.py
chmod 644 .htaccess
chmod 644 .env
```

### ШАГ 4: Перезапуск приложения

После настройки перезапустите Python приложение в панели Spaceship.
