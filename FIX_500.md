## 🐛 Если ошибка сохраняется:

### Проверьте логи

В панели Spaceship найдите логи и посмотрите на последние ошибки. Типичные проблемы:

1. **ModuleNotFoundError** - зависимости не установлены
   - Решение: нажмите "Run Pip Install"

2. **Database connection error** - проблема с БД
   - Решение: проверьте `DATABASE_URL` в переменных окружения

3. **Import error** - проблемы с путями
   - Решение: проверьте что Entry Point указан как `wsgi:application`

4. **Permission denied** - проблемы с правами доступа
   - Решение: через SSH выполните `chmod 755 .` и `chmod 777 uploads`

### Альтернативная диагностика

Если в логах нет информации, попробуйте:

1. Проверить что Python версия правильная (3.11)
2. Убедиться что все файлы загружены на сервер
3. Проверить что `wsgi.py` находится в корне проекта
4. Убедиться что структура папок правильная:
   ```
   zakup.one/
   ├── wsgi.py
   ├── app/
   │   ├── main.py
   │   └── ...
   ├── requirements.txt
   └── .env (опционально, если переменные не в панели)
   ```

## 📞 Если ничего не помогает:

1. Скопируйте полный текст ошибки из логов
2. Проверьте что все переменные окружения добавлены правильно
3. Убедитесь что зависимости установлены без ошибок
4. Проверьте что Entry Point указан как `wsgi:application` (не `wsgi.py`!)
