# 🔍 Как найти и проверить .env файл

## ✅ Это нормально - файлы с точкой скрыты!

Файлы, начинающиеся с точки (`.env`, `.htaccess`), часто скрыты по умолчанию в файловых менеджерах.

---

## 🔎 Способ 1: Показать скрытые файлы в File Manager

### В большинстве панелей управления:

1. **Найдите настройки отображения**
   - Ищите кнопку "Show Hidden Files" или "Показать скрытые файлы"
   - Или иконку с глазом 👁️
   - Или галочку "Show dot files"

2. **Включите отображение скрытых файлов**
   - Поставьте галочку или нажмите кнопку
   - Файл `.env` должен появиться в списке

### В cPanel:
- В File Manager нажмите "Settings" (Настройки)
- Поставьте галочку "Show Hidden Files (dotfiles)"
- Нажмите "Save"

### В других панелях:
- Ищите опцию "View" → "Show Hidden Files"
- Или "Settings" → "Display hidden files"

---

## 🔎 Способ 2: Проверьте через поиск

1. **Используйте поиск в File Manager**
   - Нажмите Ctrl+F или Cmd+F
   - Введите: `.env`
   - Файл должен найтись

---

## 🔎 Способ 3: Проверьте через SSH (если доступен)

```bash
cd /home/kdlqemdxxn/zakup.one
ls -la | grep .env
```

Или просто:
```bash
ls -la
```

Команда `ls -la` показывает ВСЕ файлы, включая скрытые.

---

## 🔎 Способ 4: Проверьте что файл существует

### Через терминал в панели:

```bash
cd /home/kdlqemdxxn/zakup.one
test -f .env && echo "✅ .env существует" || echo "❌ .env не найден"
```

Или:
```bash
cat .env
```

Если файл существует, вы увидите его содержимое.

---

## ✅ Проверка содержимого .env

### Если файл найден, проверьте содержимое:

```bash
cat /home/kdlqemdxxn/zakup.one/.env
```

Должно быть что-то вроде:
```
DATABASE_URL=postgresql://...
SECRET_KEY=...
DEBUG=False
CORS_ORIGINS=...
```

---

## 🆘 Если файл действительно не создан:

### Создайте через терминал:

```bash
cd /home/kdlqemdxxn/zakup.one
nano .env
```

Вставьте содержимое:
```env
DATABASE_URL=postgresql://user:password@localhost:5432/zakup_one
SECRET_KEY=ваш-секретный-ключ-минимум-32-символа
DEBUG=False
CORS_ORIGINS=["https://zakup.one","https://www.zakup.one"]
UPLOAD_DIR=/home/kdlqemdxxn/zakup.one/uploads
DOWNLOADS_DIR=/home/kdlqemdxxn/zakup.one/downloads
```

Сохраните: `Ctrl+O`, `Enter`, `Ctrl+X`

---

## 💡 Важно:

1. **Файл может быть скрыт, но существовать** - это нормально
2. **Главное - что приложение его видит** - проверьте через логи
3. **Если приложение работает - файл на месте!**

---

## ✅ Быстрая проверка:

Если вы не видите файл, но хотите убедиться что он есть:

1. **Попробуйте открыть его напрямую через URL** (не сработает, но покажет ошибку 403/404 если нет)
2. **Проверьте логи приложения** - если .env не найден, будут ошибки
3. **Используйте терминал** - самый надежный способ

---

## 🎯 Главное:

**Если файл создан правильно, приложение его найдет автоматически!**

Не обязательно видеть его в File Manager - главное что он существует на сервере.
